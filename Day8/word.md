# Day8: What's the rationale behind...?

- **カテゴリ**: 議論の型
- **意味**: 〜の根拠・理由は何ですか？
- **ニュアンス**: 単に「なぜ？」と聞くより丁寧で建設的な印象を与える。相手の判断や決定の背景にある論理的な理由を理解したいときに使う。批判ではなく、純粋に理解を深めたいという姿勢を示せる。
- **使用シーン**: コードレビュー、設計議論、技術選定の場面で、特定の実装や選択の理由を確認したいとき
- **例文**:
  - What's the rationale behind using a microservices architecture for this project?
  - What's the rationale behind choosing Redis over Memcached?
  - What's the rationale behind deprecating this API endpoint?

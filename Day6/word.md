# Day6: Let's go with...

- **カテゴリ**: 意思決定
- **意味**: ～で進めましょう、～を採用しましょう
- **ニュアンス**: 複数の選択肢の中から特定の案を選んで進めることを提案する表現。チーム内で合意を形成し、次のアクションを明確にする際に使う。
- **使用シーン**: 技術選定、設計の方向性決定、実装方針の決定など、複数の選択肢から1つを選ぶとき
- **例文**:
  - Let's go with TypeScript for this project.
  - Let's go with the second approach since it's more scalable.

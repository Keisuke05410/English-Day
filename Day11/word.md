# Day11: Are we all on the same page?

- **カテゴリ**: 合意形成の型
- **意味**: 全員が同じ認識を持っていますか？
- **ニュアンス**: チームメンバー全員が議論の内容や決定事項について共通の理解を持っているかを確認する表現。ミーティングの途中や終わりに、認識のズレがないかをチェックする際に使う。
- **使用シーン**: 要件定義の確認時、技術選定の議論後、タスクの方針決定後など、チームで重要な決定をした後の確認
- **例文**:
  - "Before we move on, are we all on the same page about the API design?"
  - "So we're going with microservices architecture. Are we all on the same page?"
  - "I just want to make sure we're all on the same page regarding the deadline."

# Day3: Does that make sense?

- **カテゴリ**: 合意形成の型
- **意味**: 理解できましたか？/ 納得できましたか？
- **ニュアンス**: 説明や提案をした後に、相手が内容を理解しているか、論理的に納得しているかを確認するカジュアルな表現。「Do you understand?」より柔らかく、相手を試すような印象を与えない。
- **使用シーン**: 技術的な説明の後、設計方針の共有時、コードレビューでの指摘時など
- **例文**:
  - "We're using a message queue to handle async processing. Does that make sense?"
  - "I think we should split this into two microservices. Does that make sense to you?"

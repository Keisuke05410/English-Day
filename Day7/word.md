# Day7: Just to confirm...

- **カテゴリ**: 定型表現（phrase）
- **意味**: 確認させてください / 念のため確認ですが
- **ニュアンス**: 重要なポイントや認識のズレがないか確認する際に使う丁寧な表現。会話の流れを止めずに、スムーズに確認できる。相手との認識を揃えたいときや、誤解を防ぎたいときに便利。
- **使用シーン**: ミーティング、仕様確認、実装前の最終確認、メール・チャットでの認識合わせ
- **例文**:
  - Just to confirm, we're deploying this feature next Monday, right?
  - Just to confirm my understanding — the API should return JSON format?
  - Just to confirm, you want me to merge this PR into the develop branch?
